## Railsのセットアップ
- rails new miniblog_2 新規アプリケーションの作成
- Gemfileの編集
  - gem 'simple_form' 
  - gem 'haml-rails'
  - gem 'pagy'
  - gem 'devise'
  - gem 'bullet', group: 'development'
  - 追加後 bundle install でインストール
  - インストール 
    - rails generate simple_form:install --bootstrap
    - rails g bullet:install
